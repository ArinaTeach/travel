# Курс «Английский для путешествий» — лендинг на React + Vite + Tailwind CSS

## Запуск

```bash
npm install
npm run dev
```

Затем откройте ссылку из терминала (обычно `http://localhost:5173`).

Проект адаптивный и оптимизирован под мобильные устройства и компьютер.
Кнопка **«Записаться на курс»** находится внизу страницы.
